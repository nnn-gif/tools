# Formatho Branding Assets

This directory contains all official Formatho brand assets and guidelines.

## Directory Structure

```
branding/
├── README.md                 # This file
├── brand_guidelines.md       # Complete brand guidelines
└── assets/
    ├── logo.png             # Primary logo (binary icon)
    ├── favicon.png          # Website favicon
```

## Asset Specifications

### Logo & Favicon

- **File:** `logo.png`, `favicon.png`
- **Format:** PNG
- **Design:** Binary numbers "01" / "10" in white on black
- **Usage:** Website sidebar, browser tab, app icons
- **Sizes:** Optimized for 16px to 512px

### Twitter Profile Photo

- **File:** `twitter_profile.png`
- **Dimensions:** 400x400px
- **Format:** PNG
- **Design:** Binary icon matching website logo
- **Usage:** Twitter profile picture (@heyformatho)

### Twitter Header

- **File:** `twitter_header.png`
- **Dimensions:** 1500x500px
- **Format:** PNG
- **Design:** Black background with "Formatho" branding and tool icons
- **Usage:** Twitter header banner

## Usage Guidelines

### ✅ Do:

- Use the binary icon as-is
- Maintain black and white color scheme
- Keep square aspect ratio for logos
- Use high-resolution versions for print

### ❌ Don't:

- Modify the binary numbers
- Add colors or gradients
- Distort or rotate the logo
- Add effects or shadows
- Use on busy backgrounds without proper contrast

## File Formats

All assets are provided in PNG format with transparent or solid backgrounds as appropriate. For vector versions or other formats, please refer to the design source files.
